# Day 08 - CSV Data Analysis

## Overview

This project demonstrates how to analyze CSV datasets using Pandas. The project performs statistical analysis, filtering, sorting, and data exploration techniques commonly used in Data Science and Machine Learning.

---

## Topics Covered

- Reading CSV Files
- Dataset Exploration
- Statistical Analysis
- Filtering Data
- Sorting Records
- Finding Top Performers

---

## Technologies Used

- Python 3
- Pandas

---

## Installation

```bash
pip install pandas
```

## Run

```bash
python main.py
```

## Learning Outcomes

- Working with CSV Files
- Data Exploration
- Statistical Analysis
- Data Filtering and Sorting
- Preparing Data for ML
